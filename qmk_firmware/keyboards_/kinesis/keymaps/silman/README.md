# Silman Kinesis Contoured/Advantage Keymap

Uses Stapelberg mod hardware but ported to KiCAD by Silman.

Keymap heavily inspired by TuesdayJohn's kinesis keymap on QMK repository.

## Features
Some features of my keymap:
- LEDs indicate certain most used layers
  - Red LED for Game Layer
  - Orange LED for Advanced Layer
  - Green LED on when momentary Function layer is active
  - Blue LED for MacOS layer
- MacOS layer swaps GUI and Ctrl
- Advanced layer allows for tap-hold momentary access to functions/numbers layer
- Function layer for media/mouse control
- Numbers/Symbols Layer
- Keypad Layer
- Game layer shifts keys over on left had so WASD is in resting position.
- Double tap shift for caps lock
- Mocking Mode
  - Toggle on/off using Adjust Layer + `6` Key
  - It WiLl MaKe AlL yOuR tExT lOoK lIkE tHiS!

In order for the LEDs to behave correctly you *must* compile using my keyboard files (hence the `-kb kinesis/silman`) as otherwise caps lock (and other "locks") may take over control of the LEDs.
